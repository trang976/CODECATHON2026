Assumption we're making: the brief's example threshold (30,000 streams) reads like a cross-platform popularity count, not one person's personal play count — no single listener here crosses that alone. Recommendations

What This Might Suggest Next

Two different kinds of "recommendation" are mixed together on purpose below — labeled honestly, not blended.

Echoprint is a hackathon prototype built on one real Spotify export (2013–2024, ~148,700 plays). The name you entered is a label for this session only — it is not stored, and every listener sees the same underlying history. Merch, ticket, and concert content is placeholder copy for demo purposes; the "recommended artists" lists are computed from real same-day co-occurrence in this listening history. Built to prompt a conversation about what a listening history alone reveals — and what it shouldn't be used to decide.
